Afinal, como vota Curitiba?

Essa pergunta surgiu enquanto Giovani Sella e Rodrigo Matana produziram a reportagem Bancada negra ainda é minoria na Câmara Municipal de Curitiba.

Para responder essa questão, foram a fundo nos dados eleitorais do TSE. Depararam-se com um banco de dados vasto, que não havia sido explorado por veículos jornalísticos. Os dados são públicos, divulgados todos os anos pelo TSE, porém não são de fácil interpretação. Por se tratar de uma base extensa, é necessário certo conhecimento em análise de dados e programação.

Como vota Curitiba surge como uma maneira de democratizar a interpretação desses dados. É uma plataforma para que jornalistas, cientistas políticos e estudiosos encontrem padrões e compreendam os fenômenos das eleições curitibanas.
